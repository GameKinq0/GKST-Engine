# 🛠️ Getting Started - Installation & Setup

This page will guide you through installing and setting up the GKST Asset Engine add-on in Blender.

## System Requirements

Before installing, make sure you have:

- **Blender 3.6 or higher** ([Download Blender](https://www.blender.org/download/))
- **Python 3.9+** (Usually included with Blender)
- **Windows, macOS, or Linux**

## Installation

Follow these step-by-step instructions to install GKST Asset Engine:

### Step 1: Download the Add-on
1. Go to the [Releases](https://github.com/GameKinq0/GKST_Engine/releases) page
2. Download the latest version: `GKST_Asset_Engine.zip`
3. **Do NOT extract the ZIP file** - Blender will handle it automatically

### Step 2: Install in Blender
1. Open **Blender**
2. Navigate to `Edit` → `Preferences` (or `Blender` → `Preferences` on macOS)
3. Click on the **Add-ons** tab on the left sidebar
4. Click the **Install...** button in the top right
5. Select the downloaded `GKST_Asset_Engine.zip` file
6. Click **Install Add-on**

### Step 3: Enable the Add-on
1. Search for "**GameKing**" in the Add-ons search bar
2. Find **3D View: GameKing Asset Engine**
3. **Check the checkbox** next to it to enable the add-on
4. You should see a notification: "**Add-on registered successfully**"

### Step 4: Open the Toolkit
1. Go to the **3D Viewport** (where you model)
2. Press `N` on your keyboard to open the right sidebar
3. Look for the **"GKST Toolkit"** panel
4. If it's not visible, look for the tabs on the right and click the GKST tab

## Verification

If you see the GKST Toolkit panel with these options, you're ready to go:
- ✅ Engine Selection Dropdown
- ✅ Smart Export Button
- ✅ Settings Panel

## Troubleshooting Installation

**Problem:** Add-on doesn't appear in the list
- **Solution:** Make sure you downloaded the correct `.zip` file and didn't extract it

**Problem:** "Add-on registration failed" error
- **Solution:** Your Blender version might be too old. Update to Blender 3.6 or newer

**Problem:** GKST Toolkit panel not showing
- **Solution:** Press `N` in the 3D viewport, or check the right sidebar tabs
